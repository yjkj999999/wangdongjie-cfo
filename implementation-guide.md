## 阶段二：建设（需要你亲手操作 🔧）

### 2.1 部署个人官网（推荐方案）

**方案A：GitHub Pages（免费，推荐）**
```
1. 注册 GitHub 账号 → 创建仓库 wangdongjie.github.io
2. 将所有文件推送到仓库 main 分支
3. 访问 https://wangdongjie.github.io/ 即可看到效果
```

**方案B：阿里云OSS + CDN（国内访问快，约¥10/月）**
```
1. 购买域名（推荐 wangdongjie.com 或 wangdongjie.cn）
2. 在阿里云开通OSS，上传全部文件
3. 开启CDN加速，绑定域名
4. 开启HTTPS（免费证书）
```

**方案C：Vercel（全球加速，免费）**
```
1. 注册 Vercel 账号 → 导入项目目录
2. 自动部署 → 绑定自定义域名
3. 自动开启HTTPS
```

### 2.2 提交搜索引擎收录

**百度站长平台：**
```
1. 注册百度站长平台 → 添加站点（你的域名）
2. 验证站点所有权（DNS验证/CNAME验证/文件验证）
3. 提交 sitemap.xml 链接
4. 使用"主动推送"功能加速收录
```

**Google Search Console：**
```
1. 注册 Google Search Console → 添加站点
2. 验证所有权（推荐DNS记录验证）
3. 提交 sitemap.xml
4. 使用 URL Inspection → Request Indexing 手动提交首页
```

### 2.3 更新LinkedIn/领英

直接复制使用 `social-platform-content.md` 中的文案：
- §1.1 个人简介（中英文）
- §1.2 工作经历标题
- §1.3 技能标签

### 2.4 更新知乎

1. 创建/完善知乎个人主页
2. 使用 §2.1 的一句话介绍和 §2.2 的个人简介
3. 发布首篇专栏文章（从 §2.3 的建议主题中选择）
4. 在个人介绍中放置官网链接

### 2.5 提交百度百科

**前置条件：** 需要在新闻媒体、行业报告中有2-3篇独立报道作为引用来源

**操作步骤：**
```
1. 联系1-2家财经/行业媒体发布人物专访或项目报道
   - 建议方向：科创板IPO亲历者、中国首例LBO复盘、业财融合实战
   - 推荐媒体：证券时报、经济观察报、36氪、界面新闻、华尔街见闻
2. 收集媒体报道的URL作为引用来源
3. 在百度百科提交词条，使用 social-platform-content.md §三 的草稿
4. 等待审核（通常2-4周）
```
